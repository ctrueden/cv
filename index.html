Curtis Rueden

Software architect, martial artist, gamer.

Code » Profile » Resume »

My name is Curtis Rueden, and I am a software developer from Madison, WI, USA. I have been programming since 1988, a Java developer since 1997, and a Linux geek since 2003.

Professionally, I oversee the software development efforts of the Laboratory for Optical and Computational Instrumentation at the University of Wisconsin-Madison. Projects in which I am involved are many, and include ImageJ, Bio-Formats, the Open Microscopy Environment, VisBio and VisAD.

Technologies I know and love: Java, Perl, Flex, PHP, Drupal, bash, GNU/Linux, OS X, XML, HTML, CSS, Eclipse, vim, Maven, Git
Technologies I know less well but want to learn more about: HTML5, Javascript/AJAX, Python, Ruby, Android
Technologies I use and often tolerate: C/C++, MATLAB, Ant, Subversion
Obsolete technologies of which I am fond: QBasic, Pascal, Mac OS Classic, MS-DOS and Windows ;-).

If you wish to contact me, you can do so at ctrueden@gmail.com.
